# Prompt Examples

## 1. 割り振られたタスクを分解する

```text
このタスクを新人が自分で進められるように整理してください。
目的、成果物、完了条件を書いてください。
自分が着手できる小タスクに分解してください。
分からないことは確認事項として分けてください。
```

## 2. 全体像が見えない時

```text
このタスクは小さい単位で割り振られていて、全体像が見えていません。
このタスクの前後工程を推測しすぎずに整理してください。
分からない部分は、確認すべきこととして一覧にしてください。
```

## 3. 今日やることを決める

```text
このタスク分解メモをもとに、今日やることを3つに絞ってください。
今すぐ着手できる順で並べてください。
```

## 4. 報告用の事実整理をする

```text
この作業内容を、作業報告テンプレートに埋めやすいように整理してください。
自由文の報告は作らず、今日やったこと、完了したこと、未完了、確認したいこと、次にやることに分けてください。
```
