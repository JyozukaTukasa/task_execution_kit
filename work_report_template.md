# 作業報告テンプレート

このテンプレートは、**事実だけを短く報告するための固定フォーマット**です。  
AI に文章を作らせてそのまま送らず、本人が内容を確認して埋めてください。

## 報告テンプレート

```text
件名:
[作業報告] <タスク名>

1. 今日やったこと
- 
- 

2. 完了したこと
- 

3. まだ終わっていないこと
- 

4. 詰まっていること・確認したいこと
- 

5. 次にやること
- 

6. 補足
- 参考資料:
- 関連ファイル:
```

## 書く時のルール

- 書いた事実だけを書く
- 推測や感想は入れすぎない
- 「やったこと」と「終わったこと」を分ける
- 詰まっていることは、何が分からないかまで書く
- 次にやることは 1 から 3 個に絞る
